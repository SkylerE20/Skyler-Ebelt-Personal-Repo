# Academic Projects & Research

This repository contains a collection of coursework and research code from my graduate studies at Georgia Institute of Technology, where I'm pursuing an MS in Computer Science with a concentration in Machine Learning.

## Contents

This repository includes:

- **Machine Learning coursework** - Implementation and analysis of supervised learning, unsupervised learning, and reinforcement learning algorithms
- **Masters research project** - Analysis of student engagement and intervention effectiveness in online learning environments (Georgia Tech edX platform)
- **Data analysis projects** - Large-scale data processing, statistical analysis, and predictive modeling

## Technical Stack

Projects utilize Python, R, SQL, Apache Spark, PostgreSQL, scikit-learn, TensorFlow/Keras, and various data analysis libraries.

## Note

Code is organized by course/project and represents work completed between 2024-2025. Some datasets are not included due to size or privacy restrictions.
