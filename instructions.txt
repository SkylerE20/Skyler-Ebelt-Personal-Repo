**Part 1**
Download php from https://www.php.net/
add php to your PATH
Download composer from https://getcomposer.org/

Download node.js

when in the folder burdellsdogsproject, make sure to run npm install

when you type npm run dev, you should see the default screen for a project

**Part 2**
configure Laravel to use your postgres db by 
1. enabling postgres in your php.ini file

in your file system, click view->show-> enable file name extensions
make sure you edit the php.ini and not php.ini-prod or php.ini-dev

uncomment the following:
extension=fileinfo
extension=pdo_pgsql
extension=pgsql
extension=zip

after you make your edits and save, run php --ini
if you run php -m, you should see the extensions that you added in the list

Go to the burdellsdogsproject folder
2. run composer install

3. run php artisan key:generate
also run php artisan migrate
restart pgadmin if you have it open

4: edit the .env file with your database info

to run the setup:
open two terminals
in one terminal run: php artisan serve
in another terminal run: npm run dev
